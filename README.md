A JavaScript project that dynamically generates YouTube-style video cards.

## 📌 About the Project

This project demonstrates how JavaScript can be used to dynamically generate YouTube-style video cards using video metadata such as title, channel name, views, duration, and thumbnail. It showcases DOM manipulation, template literals, and dynamic content rendering to create an interactive and visually appealing user interface.

## 🚀 Technologies & Concepts Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Template Literals
- Dynamic Content Rendering
- Responsive UI Design
